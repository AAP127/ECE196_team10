Gotta Park Fast – Parking Spot Status

A smart solution for real-time parking spot availability at UCSD

About the Project

This website is part of our UCSD ECE 196 senior design project. We are developing a smart parking monitoring system using proximity sensors and microcontrollers. This web interface will show parking spot availability. The current version is static for presentation purposes.
